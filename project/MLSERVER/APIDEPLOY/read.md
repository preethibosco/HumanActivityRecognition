### How to Run the Server
---

Please run 

```
 uvicorn app:app
```
from the folder where you have the app.py

```
PS C:\swararoy\work\aiml\aiml-codebase\capstone\models\logistic-regression> uvicorn app:app
INFO:     Started server process [85984]
INFO:     Waiting for application startup.
INFO:     Application startup complete.
INFO:     Uvicorn running on http://127.0.0.1:8000 (Press CTRL+C to quit)
```

and use a REST client to send the Human Activity Record data as a JSON as POST request to 
http://127.0.0.1:8000/har/predict
and get a response like

```
{
	"prediction": "STANDING"
}
```
